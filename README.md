Repositorio para experimentar la migración de la librería xAct de Wolfram a otro lenguaje abierto

- Instalar [Wolfram Engine](https://www.wolfram.com/engine/)
- Descargar e instalar [xAct](http://xact.es/download.html#install)

- Instalar Julia
- Instalar [MathLink](https://github.com/JuliaInterop/MathLink.jl)

- Instalar Python
- Instalar [Wolfram Client for Python](https://reference.wolfram.com/language/WolframClientForPython/)
